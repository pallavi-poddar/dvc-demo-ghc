## Getting Started

Read the docs

### Prerequisites


```
  install git
  install dvc
   ```

### Installation




1. Clone the repo
   ```sh
   $ git clone https://github.com/pallavi-poddar/DVC-demo.git
   $ cd DVC-demo
   ```

2. Install Virtual Environment and install dependencies
   ```sh
   $ python3 -m venv omni_env
   $ source omni_env/bin/activate
   $ pip install -r requirements.txt
   
   ```
